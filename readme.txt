== Description ==
This WordPress plug-in allows you to copy Wordpress posts to other Wordpress websites.
The plugin comes with its own 'Settings' interface, you can add multiple domains, ftp and database details (the additional website database should be "remotely allowed").

= General Features =
* Copy post to single domain.
* Copy post to selected domains.
* Copy multiple posts to all domains (single click).
* Single interface to manage multiple Wordpress blogs.
* Lightweight
* Works well with other plug-ins with no conflict issues.

== Installation == 

The automatic plug-in installer should work for most people. Manual installation is easy and takes less than two minutes.
1. Download the plug-in, unpack it and upload the 'migrate-posts' folder to your wp-content/plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Migrate Post Domains -> 'Add New' Page to add the new domain details.
4. Go to Migrate Post Domains Page to list out all added domains.

To use the plugin, you need to follow below instructions:
* Added Wordpress site domains database tables prefix should be exactly same as per your installed plugin domain.
* All database and ftp details should be successfully connected on added domains.
* Database should be allowed remotely accessible.

== Frequently Asked Questions ==
How to use this for migrate the post to different Wordpress site domains?

Go to the 'Migrate post domains', select domain where you want to migrate the post then select 'migrate posts' from drop down and press 'apply' button. Here, you need to select post and click on submit button.
